# Level 5 — Korean Stock Perp Market Making

> Last modified: 2026-03-27

---

## Overview

Lighter DEX의 한국 주식 Perp은 아직 **초기 시장**으로 유동성이 매우 얇다. 이 비효율적 시장에서 마켓메이커로 활동하여 매수-매도 스프레드를 수취하는 전략. 기관 마켓메이커가 진입하기 전 **선발주자 이점(first-mover advantage)**이 존재.

---

## Strategy 5-A: Bid-Ask Spread Capture

### Mechanism
유동성이 얇은 시장에서 양방향 호가(bid/ask)를 제시하고 스프레드를 수익화.

### Market Structure Analysis
```
성숙 시장 (BTC/ETH Perp):
├── Bid-Ask Spread: 0.01~0.05%
├── 마켓메이커 수: 수백 개
├── 경쟁: 극심
└── 수익 마진: 극박

초기 시장 (한국 주식 Perp on Lighter):
├── Bid-Ask Spread: 0.5~2.0% (추정)
├── 마켓메이커 수: 극소수 (또는 없음)
├── 경쟁: 미약
└── 수익 마진: 높음
```

### Execution
```
삼성전자 Perp 현재 중간가: $50,000

마켓메이킹 호가:
- Bid (매수): $49,750 (-0.5%)
- Ask (매도): $50,250 (+0.5%)
- Spread: $500 (1.0%)

체결 시나리오:
1. Taker가 Ask에 매수 → 내가 $50,250에 매도
2. 가격 하락 → Taker가 Bid에 매도 → 내가 $49,750에 매수
3. 순이익: $500 (스프레드 1.0%)
```

### Return Profile
| 항목 | 수치 |
|------|------|
| 평균 스프레드 | 0.5~2.0% |
| 일일 회전 | 2~5회 |
| 일일 수익 (자본 $100K) | $100~$1,000 |
| **연간 기대 수익** | **20~60%** |

### Critical Risks
| 리스크 | 심각도 | 설명 |
|--------|--------|------|
| **재고 리스크** | 고 | 한쪽 방향으로만 체결 → 방향성 노출 |
| **갭 리스크** | 중 | KRX 개장 시 큰 갭 발생 → 포지션 손실 |
| **유동성 고갈** | 중 | 거래량 급감 시 호가 체결 불가 |
| **스마트컨트랙트** | 중 | Lighter DEX 자체 리스크 |

### Inventory Management
```
규칙 1: Net Position Limit
- 순 포지션 ≤ 자본의 20%
- 초과 시 한쪽 호가 철회

규칙 2: 시간대별 스프레드 조정
- KRX 장중 (09:00~15:30): 스프레드 축소 (0.3~0.5%) — 가격 참조 가능
- KRX 폐장 (15:30~09:00): 스프레드 확대 (0.8~2.0%) — 불확실성 반영
- 주말: 스프레드 최대 (1.5~3.0%)

규칙 3: 델타 헤지
- 순 포지션이 한쪽으로 쏠리면 KRX 현물/선물로 헤지
- 단, KRX 폐장 시 헤지 불가 → 포지션 규모 축소
```

## Strategy 5-B: LP Vault Participation

### Mechanism
Lighter DEX의 LLP(Liquidity Provider) Vault에 자금을 예치하여 프로토콜 레벨에서 마켓메이킹에 참여. 직접 호가 관리 없이 자동화된 수익.

### Return Data
| 플랫폼 | APY | 메커니즘 |
|--------|-----|---------|
| Lighter LLP Vault | ~60% (2025 초) | 거래 수수료 + 펀딩비 + 청산 수익 |
| Hyperliquid HLP | 6~8% | 거래 수수료 + 스프레드 |
| GMX V2 GLV | 10~20% | 거래 + 차입 + 청산 수수료 |

### Risk
- **LP는 트레이더의 상대방**: 트레이더가 수익 → LP 손실
- 변동성 급등 시 LP 자본 크게 감소 가능
- 보험 기금 소진 시 LP가 최종 흡수자

## Evaluation (Level 5)

| 기준 | 5-A (직접 MM) | 5-B (LP Vault) |
|------|-------------|---------------|
| 실현가능성 | 7 | 9 |
| 수익성 | 9 | 8 |
| 리스크 수준 | 4 | 4 |
| 자본 효율성 | 8 | 6 |
| 확장성 | 4 | 5 |
| 규제 리스크 | 6 | 6 |
| **총점** | **38/60 (B)** | **38/60 (B)** |

### Verdict
**수익성 > 리스크**: YES (단, 리스크 관리 필수)
- 초기 시장의 넓은 스프레드 = 높은 수익 잠재력
- **최대 위험**: 재고 리스크 — 한쪽 방향으로 포지션 쏠림 시 큰 손실
- 자동화된 재고 관리 + 포지션 한도가 핵심
- **시간 제한 기회**: 기관 MM 진입 전까지 6~12개월 추정
- LP Vault는 수동적이지만 높은 APY (60%) 매력적 — 다만 LP 손실 리스크 이해 필수
