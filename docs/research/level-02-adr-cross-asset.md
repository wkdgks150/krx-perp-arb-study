# Level 2 — ADR/GDR & Cross-Asset Correlation Arbitrage

> Last modified: 2026-03-27

---

## Overview

한국 대형주는 KRX 외에도 ADR(미국), GDR(유럽) 형태로 글로벌 거래소에서 거래된다. 이 복수 시장의 가격 차이와, 상관관계 높은 자산 간의 일시적 괴리를 활용하는 전략.

---

## Strategy 2-A: Samsung ADR/GDR Arbitrage

### Mechanism
삼성전자는 KRX(원화), LSE GDR(달러), 그리고 Lighter DEX Perp(USDC)에서 동시에 거래된다. 세 시장의 가격이 완벽하게 일치하지 않으며, 특히 **타임존 차이**에서 괴리가 발생한다.

```
[KRX 삼성전자] ←→ [LSE Samsung GDR] ←→ [Lighter SAMSUNG Perp]
   09:00-15:30 KST    08:00-16:30 GMT       24/7
       원화                달러/GBP            USDC
```

### Execution
```
Scenario: 글로벌 반도체 호재 발생 (미국 장중)

1. LSE GDR 상승 +2% (유럽 장중)
2. Lighter Perp 상승 +1.8% (24/7, 약간 지연)
3. KRX는 폐장 상태 → 다음날 반영

Action:
- Lighter Perp 롱 진입 (아직 0.2% 할인)
- 또는 KRX 개장 시 현물 매수 준비
- GDR-Perp 스프레드 0.2% 수렴 시 청산
```

### Return Profile
| 항목 | 수치 |
|------|------|
| ADR/GDR 가격 괴리 | 20~100 bps (평균) |
| 환율 변동 추가 영향 | ±50 bps |
| 거래당 기대 수익 | 0.2~1.0% |
| 연간 기회 횟수 | 50~100회 |
| **연간 기대 수익** | **10~30%** (적극적 실행 시) |

### Risks
| 리스크 | 심각도 | 설명 |
|--------|--------|------|
| 환율 리스크 | 중 | KRW/USD 변동이 수익 잠식 |
| 결제 시차 | 중 | KRX T+2, GDR 결제 별도 |
| 유동성 | 중 | GDR은 KRX 대비 유동성 낮음 |
| 배당세 차이 | 저 | ADR/GDR 배당 원천징수율 차이 |

## Strategy 2-B: Correlated Sector Pair (반도체 섹터)

### Mechanism
반도체 섹터 내 상관관계가 높은 자산들의 일시적 괴리를 활용.

```
상관 그룹:
- 삼성전자 ↔ SK하이닉스 (메모리)
- KOSPI 200 ↔ SOX (필라델피아 반도체 지수)
- 삼성전자 ↔ TSMC (파운드리 연관)
- SK하이닉스 ↔ Micron (HBM 경쟁)
```

### Execution
```
Scenario: Micron 실적 발표 → HBM 수요 상향

1. Micron +5% (미국 장중)
2. SK하이닉스 Perp (Lighter): 아직 +2% (비효율적 반영)
3. 삼성전자 Perp: +1% (간접 수혜, 더 느리게 반영)

Action:
- SK하이닉스 Perp 롱 진입
- 삼성전자 Perp 롱 진입 (헤지로 KOSPI 200 숏 가능)
- KRX 개장 시 수렴 확인 후 청산
```

### Return Profile
| 항목 | 수치 |
|------|------|
| 섹터 내 상관 괴리 | 0.5~2% |
| 수렴 시간 | 수 시간~1일 |
| 연간 기회 | 30~60회 |
| **연간 기대 수익** | **8~20%** |

### Risks
- **상관관계 붕괴**: 기업별 이슈로 섹터 상관 깨질 수 있음
- **방향성 리스크**: 완전한 델타뉴트럴이 아닌 경우 존재
- **정보 비대칭**: 글로벌 투자자 대비 정보 접근 지연

### Verdict
**수익성 > 리스크**: YES (조건부)
- ADR/GDR 괴리는 구조적으로 반복 발생 → 안정적 기회
- 환율 헤지를 병행하면 리스크 대비 수익 우위 확보 가능
- KRX 타임 갭이 핵심 엣지 — 글로벌 뉴스가 KRX에 반영되기 전 Perp에서 선취
