# Small Capital High-Rotation Playbook

> Last modified: 2026-03-27
> 대상: $1K~$10K 소자본, 회전 극대화

---

## 왜 연간 수익률이 낮아 보였는가

기존 Level 1~10 평가는 **기관 관점 (대자본, 보수적 레버리지, 연간 기준)** 으로 작성되었다.
소자본에서는 관점을 바꿔야 한다:

```
기관: $1M, 2x 레버, 연 15% = $150,000 ← 금액은 크지만 %는 낮음
소자본: $5K, 5x 레버, 일 0.5% × 365일 = 연 500%+ ← 복리 효과
```

**김치프리미엄의 본질** = 같은 자본으로 "사고 → 옮기고 → 팔고"를 **반복**
**Perp DEX의 본질** = 같은 자본으로 "진입 → 펀딩 수취 → 청산 → 재진입"을 **반복**

차이점: 김프는 방향성 리스크 있음. 펀딩 차익은 **델타뉴트럴** (방향 무관).

---

## 소자본 핵심 공식

```
일일 수익 = 포지션 크기 × 펀딩레이트 × 일일 횟수
포지션 크기 = 자본 × 레버리지
복리 효과 = (1 + 일일수익률) ^ 일수
```

## Scenario: $5,000 시작

### Setup A: Conservative (3x 레버)

```
자본: $5,000
레버리지: 3x → 포지션 $15,000
펀딩레이트: 0.05% / 8h (평균)
일일 횟수: 3회

일일 수익: $15,000 × 0.05% × 3 = $22.50 (자본 대비 0.45%/일)
수수료: $0 (Lighter 제로피)

1개월: $5,000 × 1.0045^30 = $5,720 (+$720, +14.4%)
3개월: $5,000 × 1.0045^90 = $7,455 (+$2,455, +49.1%)
6개월: $5,000 × 1.0045^180 = $11,118 (+$6,118, +122%)
1년:   $5,000 × 1.0045^365 = $25,800 (+$20,800, +416%)
```

### Setup B: Moderate (5x 레버)

```
자본: $5,000
레버리지: 5x → 포지션 $25,000
펀딩레이트: 0.05% / 8h
일일 횟수: 3회

일일 수익: $25,000 × 0.05% × 3 = $37.50 (자본 대비 0.75%/일)

1개월: $5,000 × 1.0075^30 = $6,260 (+$1,260, +25.2%)
3개월: $7,840 (+$2,840, +56.8%)  ← 복리 재투자
6개월: $12,280 (+$7,280, +145.6%)
1년:   $76,900 (!!!) ← 이론적 최대, 실제로는 자금 풀 한계로 불가능
```

### Setup C: 강세장 (높은 펀딩)

```
자본: $5,000, 5x 레버
펀딩레이트: 0.1% / 8h (강세장)
일일: $25,000 × 0.1% × 3 = $75 (1.5%/일)

1개월: $5,000 × 1.015^30 = $7,842 (+56.8%)
3개월: $18,740 (+274%)
```

## 현실 체크: 왜 모두가 안 하는가

### 1. 펀딩레이트는 고정이 아니다
```
강세장: +0.05~0.2% / 8h → 돈 번다
보합장: +0.01~0.03% / 8h → 적게 번다
약세장: -0.01~-0.05% / 8h → 역으로 지불 (손실!)

→ 약세장 진입 시 즉시 청산 필수
→ "항상 벌린다"가 아니라 "시장 상황 봐서 벌린다"
```

### 2. 청산 리스크
```
5x 레버: 가격 20% 반대 방향 → 청산
10x 레버: 가격 10% 반대 방향 → 청산

델타뉴트럴이라 가격 리스크 없다고? 아니다.
- 현물과 Perp 가격이 일시적으로 크게 괴리 가능
- Perp 쪽만 급등/급락 → 한쪽 레그 청산
- 청산되면 헤지 깨짐 → 방향성 노출
```

### 3. Lighter DEX 한국 주식 유동성
```
BTC/ETH Perp: 유동성 깊음 → 대량 진입/청산 가능
삼성전자 Perp: 유동성 얕음 → 슬리피지 클 수 있음

→ 소자본에게는 오히려 유리 (작은 규모는 유동성 영향 적음)
→ 대자본에게는 불리
```

### 4. 약세장 방어
```
핵심: 펀딩레이트가 음(-)으로 전환되면 즉시 청산

모니터링 알림 설정:
- 펀딩 < +0.01% → 주의
- 펀딩 < 0% → 즉시 청산
- 자동화 봇이 필수
```

## 소자본 최적 전략 조합

### Tier 1: 메인 엔진 (자본의 60%)

**펀딩레이트 팜 (Funding Rate Farming)**

```
$3,000 × 5x = $15,000 포지션
Lighter DEX (제로피) 삼성전자 Perp

구조:
├── KRX 현물 삼성전자 매수 (또는 대체: Lighter에서 KOSPI 롱)
├── Lighter에서 삼성전자 Perp 숏
├── 8시간마다 펀딩 수취
└── 매일 복리 재투자

일일 수익: $15,000 × 0.05% × 3 = $22.50 (0.75%/일)
```

**왜 이게 "김프 매매"와 같은가:**
```
김프: 업비트 매수 → 바이낸스 매도 → 반복 (회전당 3~10%)
펀딩팜: 숏 유지 → 8시간마다 수취 → 재투자 (회전당 0.05%)

차이: 김프는 1회전에 크지만 빈도 낮음
      펀딩팜은 1회전에 작지만 하루 3회 × 365일 = 연 1,095회전
      복리 효과로 실질 수익 김프 이상 가능
```

### Tier 2: 부스터 (자본의 30%)

**장외시간 갭 트레이딩 (Off-Hours Scalping)**

```
$1,500 × 10x = $15,000 포지션
이벤트 때만 진입 → 청산 → 대기 → 반복

일일 루틴:
├── 15:30 KRX 폐장 → 모니터링 시작
├── 글로벌 뉴스/이벤트 발생 시 Perp 진입
├── 갭 방향으로 포지션 (방향성 베팅)
├── KRX 개장 전후 청산
└── 수익: 이벤트당 0.5~3%

빈도: 월 10~20회
월 수익: 5~15% (방향성이므로 손실 가능성도 있음)
```

### Tier 3: 안전장치 (자본의 10%)

**현금 버퍼**

```
$500 예비 자본
├── 마진콜 대비 추가 증거금
├── 기회 포착 시 긴급 투입
└── 절대 전액 투입하지 않는 습관
```

## 회전 구조 시각화

```
$5,000 시작

Day 1-30: 펀딩팜 0.75%/일 복리
├── $5,000 → $6,260 (+25%)
└── 갭 트레이딩 추가: +$200~500

Day 31-90: 자본 증가 → 포지션 확대
├── $6,500 × 5x = $32,500 포지션
├── 일일 펀딩: $48.75
└── 3개월 차 자본: ~$9,000

Day 91-180:
├── $9,000 × 5x = $45,000 포지션
├── 일일 펀딩: $67.50
├── 갭 트레이딩 병행
└── 6개월 차 자본: ~$15,000

이것이 "회전할수록 돈이 남는 구조"
```
