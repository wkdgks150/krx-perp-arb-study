# Level 4 — Pairs Trading & Statistical Arbitrage

> Last modified: 2026-03-27

---

## Overview

공적분(cointegration) 관계에 있는 한국 주식 쌍의 일시적 괴리를 24/7 Perp 시장에서 포착하여 평균 회귀(mean reversion)로 수익화. 학술 연구에서 한국 주식시장에서의 페어 트레이딩 유효성이 입증됨.

---

## Strategy 4-A: Korean Stock Pair Trading (24/7 확장)

### Mechanism
KRX에서 통계적으로 검증된 주식 쌍을 Perp DEX에서 24/7 거래. 기존 KRX 6.5시간 → 24시간으로 거래 창이 확대되면서 **더 많은 괴리 기회**를 포착.

### Validated Korean Pairs
| Pair | 섹터 | 상관계수 | 공적분 | 24/7 활용도 |
|------|------|---------|--------|-----------|
| 삼성전자 ↔ SK하이닉스 | 반도체 | 0.85+ | 강 | **Lighter에서 모두 지원** |
| 삼성전자 ↔ 현대차 | 대형주 | 0.60+ | 중 | **Lighter에서 모두 지원** |
| SK하이닉스 ↔ 현대차 | 대형주 | 0.55+ | 약-중 | **Lighter에서 모두 지원** |
| KOSPI 200 ↔ 개별주 | 지수-개별 | 다양 | 강 | **Lighter에서 지원** |

### Execution
```
1. 삼성-하이닉스 스프레드 = (삼성 Perp / 하이닉스 Perp) 비율 모니터링
2. 20일 이동평균 대비 2σ 이탈 시:
   - 스프레드 확대: 저평가 종목 롱 + 고평가 종목 숏
   - 스프레드 축소: 반대 포지션
3. 평균 회귀 시 (1σ 이내) 청산

Example:
정상 비율: 삼성/하이닉스 = 4.5
현재 비율: 5.2 (+2σ 이탈) → 삼성 고평가, 하이닉스 저평가
Action: 하이닉스 롱 + 삼성 숏
수렴 시: 양쪽 청산 → 스프레드 차이만큼 수익
```

### 24/7 Advantage
```
기존 (KRX only):
├── 09:00~15:30: 거래 가능 (6.5h)
└── 15:30~09:00: 기회 놓침 (17.5h)

Perp DEX:
├── 24시간 모니터링
├── KRX 폐장 중 글로벌 뉴스로 인한 괴리 포착
├── 주말 65시간 동안 추가 기회
└── 거래 창 4.8배 확대 (168h / 32.5h)
```

### Return Profile
| 항목 | 수치 |
|------|------|
| 거래당 수익 | 0.5~3% |
| 월간 거래 횟수 | 8~15회 |
| 승률 | 55~65% |
| Sharpe Ratio | 1.2~2.0 |
| **연간 기대 수익** | **15~25%** |

## Strategy 4-B: Index vs Component Arbitrage

### Mechanism
KOSPI 200 지수 Perp과 개별 구성 종목 Perp 간의 괴리 활용. 지수는 200개 종목의 가중 평균이므로, 개별 종목이 지수 대비 과도하게 움직일 때 차익거래.

### Execution
```
삼성전자 가중치: KOSPI 200의 ~30%

Scenario: 삼성전자만 악재로 -3%, 나머지 변동 없음
KOSPI 200 Perp: -0.9% (삼성 가중치 반영)
삼성전자 Perp: -3%

삼성의 "과잉 하락분" = -3% - (-0.9% × 삼성비중 보정)

Action:
- 삼성전자 Perp 롱 (과매도 판단)
- KOSPI 200 Perp 숏 (헤지)
- 비율: 삼성 $10,000 롱 vs KOSPI $3,000 숏 (30% 가중치)
```

### Return Profile
| 항목 | 수치 |
|------|------|
| 지수-종목 괴리 | 0.3~1.5% |
| 수렴 시간 | 4~24시간 |
| 월간 기회 | 10~20회 |
| **연간 기대 수익** | **10~20%** |

## Strategy 4-C: Mean Reversion on Korean Perps (24/7)

### Mechanism
한국 주식은 2000년 이후 평균 회귀 패턴을 보인다는 연구 결과. Perp DEX에서 24시간 볼린저 밴드/MACD 기반 평균 회귀 트레이딩.

### Execution
```
삼성전자 Perp 20일 MA: $50,000
현재가: $47,500 (-2σ, 볼린저 하단 터치)

Action:
- 롱 진입 @ $47,500
- 목표: 20일 MA 복귀 ($50,000) → +5.3%
- 손절: -3σ ($46,250) → -2.6%
- Risk/Reward: 1:2

보유 기간: 4~48시간
```

### Return Profile
| 항목 | 수치 |
|------|------|
| 거래당 수익 | 0.5~3% |
| 승률 | 55~60% |
| Risk/Reward | 1:1.5~1:2.5 |
| 월간 거래 | 10~20회 |
| **연간 기대 수익** | **12~20%** |

## Combined Evaluation (Level 4)

| 기준 | 4-A (Pairs) | 4-B (지수-종목) | 4-C (Mean Rev) |
|------|------------|----------------|---------------|
| 실현가능성 | 8 | 7 | 8 |
| 수익성 | 8 | 7 | 7 |
| 리스크 수준 | 7 | 7 | 6 |
| 자본 효율성 | 7 | 6 | 7 |
| 확장성 | 5 | 5 | 5 |
| 규제 리스크 | 7 | 7 | 7 |
| **총점** | **42/60 (A-)** | **39/60 (B)** | **40/60 (B+)** |

### Verdict
**수익성 > 리스크**: YES (강력)
- **Level 4-A (Pairs)가 전체 레벨 중 최고 총점 42/60**
- 삼성-하이닉스 쌍이 Lighter DEX에서 모두 지원 → 즉시 실행 가능
- 학술적으로 검증된 전략 + 24/7 확장 = 거래 기회 4.8배 증가
- 시장 방향성 중립 (market-neutral) → 리스크 관리 우수
- **핵심 엣지**: KRX 폐장 중 발생하는 쌍 괴리를 다른 참여자보다 먼저 포착
