## Evaluation Framework

### 6 Dimensions (각 10점 만점, 총 60점)

| 기준 | 설명 | 평가 기준 |
|------|------|----------|
| **실현가능성** | 현재 인프라로 즉시 실행 가능한가 | 10=즉시 가능, 1=현재 불가 |
| **수익성** | 비용 차감 후 기대 순수익률 | 10=30%+, 5=10%, 1=마이너스 |
| **리스크 수준** | 원금 손실 가능성 (역수: 낮을수록 높은 점수) | 10=거의 무위험, 1=원금 전액 손실 가능 |
| **자본 효율성** | 필요 자본 대비 수익 효율 | 10=소자본 고수익, 1=대자본 저수익 |
| **확장성** | 자본 규모 증가 시 수익 유지 가능성 | 10=무제한 확장, 1=소액만 가능 |
| **규제 리스크** | 법적 문제 가능성 (역수: 낮을수록 높은 점수) | 10=완전 합법, 1=명백히 위법 |

## Individual Strategy Evaluation

### Strategy 1: Funding Rate Arbitrage (펀딩레이트 차익거래)

| 기준 | 점수 | 근거 |
|------|------|------|
| 실현가능성 | **7** | Lighter DEX에서 즉시 가능. 단, KRX 현물 매수 + DEX 연동 인프라 필요 |
| 수익성 | **8** | 평균 10~19% 연간, 강세장 30%+. 제로피로 비용 부담 최소 |
| 리스크 수준 | **7** | 델타뉴트럴이나 펀딩 반전/청산 리스크 존재. 보수적 레버리지로 관리 가능 |
| 자본 효율성 | **7** | 양측 자본 필요하나, 10x 레버리지로 Perp 측 자본 효율화 가능 |
| 확장성 | **6** | DEX 유동성 한계. 삼성전자 수준은 가능하나 소형주 불가 |
| 규제 리스크 | **5** | 외환거래법 보고의무, 스테이블코인 규제 불확실성 |
| **총점** | **40/60** | **등급: B+** |

**Verdict:** 가장 검증된 전략. 인프라만 갖추면 안정적 수익 기대. 규제 리스크가 최대 변수.

### Strategy 2: Basis Trading (베이시스 트레이딩)

| 기준 | 점수 | 근거 |
|------|------|------|
| 실현가능성 | **6** | 베이시스 모니터링 + 동시 실행 인프라 필요. 공매도 제한으로 역방향 불가 |
| 수익성 | **6** | 5~10% 연간. 스프레드 크기와 빈도에 의존 |
| 리스크 수준 | **6** | 스프레드 확대 리스크, 동시 실행 실패 리스크 |
| 자본 효율성 | **5** | 양측 자본 + 빈번한 회전 필요 |
| 확장성 | **5** | 유동성 한계 + 대규모 주문 시 스프레드 자체를 움직임 |
| 규제 리스크 | **5** | Strategy 1과 동일한 규제 환경 |
| **총점** | **33/60** | **등급: B-** |

**Verdict:** 펀딩레이트 차익의 보완 전략. 단독으로는 수익성 부족. 공매도 제한이 큰 걸림돌.

### Strategy 3: Off-Hours Price Dislocation (장외시간 가격 괴리)

| 기준 | 점수 | 근거 |
|------|------|------|
| 실현가능성 | **8** | Lighter DEX에서 장외시간 Perp 거래 즉시 가능. 인프라 요구 낮음 |
| 수익성 | **7** | 이벤트당 1~5%. 연간 횟수 제한 → 총 수익 변동 큼 |
| 리스크 수준 | **5** | **방향성 베팅** — 뉴스 해석 오류 시 손실. 델타뉴트럴 아님 |
| 자본 효율성 | **8** | 이벤트 때만 자본 투입. 비 이벤트 시 유휴 자본 다른 용도 가능 |
| 확장성 | **4** | 이벤트 빈도 제한. 대규모 자본 투입 시 유동성 문제 |
| 규제 리스크 | **6** | Perp만 거래 시 KRX 연동 불필요 → 상대적으로 단순 |
| **총점** | **38/60** | **등급: B** |

**Verdict:** KRX의 구조적 시간 갭을 가장 직접적으로 활용. 다만 방향성 리스크로 "무위험"과 거리 있음. 이벤트 선별 능력이 핵심.

### Strategy 4: Cross-Market Arbitrage (크로스마켓 차익거래)

| 기준 | 점수 | 근거 |
|------|------|------|
| 실현가능성 | **3** | **현재 불가** — 한국 주식 perp은 Lighter DEX에서만 거래 |
| 수익성 | **5** | 향후 가능 시 3~8% 연간 추정 |
| 리스크 수준 | **6** | 델타뉴트럴. 오라클 불일치가 주 리스크 |
| 자본 효율성 | **4** | 복수 DEX 마진 관리 → 자본 분산 비효율 |
| 확장성 | **3** | 플랫폼 수와 유동성에 종속 |
| 규제 리스크 | **5** | 복수 DEX 이용 시 추적/규제 복잡성 증가 |
| **총점** | **26/60** | **등급: C+** |

**Verdict:** 미래 전략. 한국 주식 perp 생태계가 확장되어야 실행 가능. 현재는 모니터링만.

### Strategy 5: Delta-Neutral Volatility (델타뉴트럴 변동성)

| 기준 | 점수 | 근거 |
|------|------|------|
| 실현가능성 | **7** | Strategy 1과 동일 인프라. 추가로 펀딩레이트 분석 로직 필요 |
| 수익성 | **7** | 강세장 20~40%, 보합장 10~15%. 약세장에서는 포지션 미진입 → 0% |
| 리스크 수준 | **7** | 델타뉴트럴 + 선택적 진입 → 리스크 관리 우수 |
| 자본 효율성 | **6** | 선택적 투입으로 Strategy 1 대비 효율적 |
| 확장성 | **6** | Strategy 1과 유사. 유동성 한계 내에서 확장 가능 |
| 규제 리스크 | **5** | Strategy 1과 동일 |
| **총점** | **38/60** | **등급: B** |

**Verdict:** Strategy 1의 스마트 버전. 조건부 진입으로 약세장 손실 회피. 분석 역량이 추가로 필요.

1. **펀딩레이트 차익거래 (40점, B+)** — 가장 높은 총점. 안정적 수익 + 검증된 메커니즘

## Critical Prerequisites (실행 전 필수 사항)

1. **법률 검토**: 외환거래법, 가상자산 규제 전문 법률 자문
2. **세무 검토**: Perp DEX 수익의 과세 구분 확인
3. **인프라 구축**: KRX 증권사 API + Lighter DEX API 연동
4. **자본 경로**: 원화 → USDC 전환 경로 확보 (합법적)
5. **모니터링 시스템**: 펀딩레이트, 마진율, 오라클 상태 실시간 감시
6. **리스크 한도**: 최대 손실 한도 설정 + 자동 청산 트리거

## Final Verdict

> **"무위험 차익거래"는 이론적으로만 존재한다.**

실행 리스크, 규제 리스크, 자본통제를 고려하면 **완전한 무위험은 불가능**하다. 그러나 **체계적 리스크 관리** 하에 **연 12~20%의 시장중립 수익**은 현실적으로 달성 가능한 목표다.

가장 큰 변수는 **규제**다. 한국 정부의 가상자산 정책 방향에 따라 전략 전체가 무효화될 수 있으므로, **규제 모니터링이 투자 전략만큼 중요하다.**
