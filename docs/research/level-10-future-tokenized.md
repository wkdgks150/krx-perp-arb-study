## Strategy 10-A: Tokenized Equity Dividend Arbitrage

### Background
- 2027년 전자증권법 개정 → 블록체인 네이티브 증권 발행 합법화
- SEC 등록 토큰화 주식은 **실제 배당권 + 의결권** 포함 (Kraken 2025 출시 버전은 배당 미포함)
- BCG 추정: 한국 토큰화 증권 시장 367조 원 ($249B)

### Mechanism
토큰화 주식의 배당락일 전후 가격 패턴을 Perp DEX와 결합.

```
배당 차익거래:

1. 배당락일 D-5: 토큰화 삼성전자 매수 (배당권 확보)
2. 동시: Lighter SAMSUNG Perp 숏 (델타 헤지)
3. 배당락일 D: 주가 배당금만큼 하락 (이론적)
   - 현물: -배당금, 그러나 배당 수취
   - Perp: +배당금 상당 수익 (숏 이익)
4. 순이익: 배당금 - (가격 미하락분) - 수수료

핵심: 주가가 배당금 전액만큼 하락하지 않는 경우가 많음
→ 배당 수취 + 불완전 하락분 = 초과 수익
```

### Return Profile
| 항목 | 수치 |
|------|------|
| 한국 대형주 배당 수익률 | 2~4% |
| 불완전 배당락 초과 수익 | 0.3~1% per event |
| 연간 배당 이벤트 | 4회 (분기별) |
| 헤지 비용 | -0.5~-1% |
| **순 연간 기대 수익** | **4~8%** |

### Prerequisites
- 토큰화 증권 거래 인가 (2027~)
- 토큰화 주식의 배당 메커니즘 확인
- 원천징수세 처리 방안

## Strategy 10-B: Tokenized Security - Perp 3중 차익거래

### Mechanism
토큰화 증권이 시행되면 동일 자산이 **3개 시장**에서 거래된다. 3중 가격 차이를 삼각 차익거래로 수익화.

```
Layer 1: KRX 현물 (원화, 09:00~15:30)
Layer 2: 토큰화 증권 (USDC/KRW, 24/7)
Layer 3: Perp DEX (USDC, 24/7)

삼각 차익거래:
KRX < 토큰화 < Perp 일 때:
1. KRX에서 현물 매수 (최저가)
2. 토큰화로 전환 (온체인)
3. Perp에서 숏 (최고가)
4. 수렴 시 청산

또는 역방향: Perp < 토큰화 < KRX
```

### Return Profile
| 항목 | 수치 |
|------|------|
| 3중 스프레드 | 0.5~3% |
| 수렴 시간 | 수 시간~1일 |
| 전환 비용 | 0.1~0.5% |
| **순 거래당 수익** | **0.3~2.0%** |
| 월간 기회 | 10~30회 |
| **연간 기대 수익** | **15~40%** |

### Why This Could Be Massive
```
2027년 초기 시장 비효율:
├── 토큰화-현물 가격 괴리 빈번 (시장 미성숙)
├── 마켓메이커 부재 → 넓은 스프레드
├── 결제 시차 (KRX T+2 vs 토큰 즉시)
├── 시간대 갭 (KRX 폐장 vs 토큰 24/7)
└── 규제 불확실성 → 기관 참여 지연

→ 개인/소규모 참여자에게 유리한 윈도우 (6~12개월)
```

## Strategy 10-C: Convertible Arbitrage (전환사채 차익거래)

### Mechanism
토큰화 전환사채가 온체인에 등장하면 (2027~2028), 전환사채 매수 + 주식 숏의 전통적 전환사채 차익거래를 DeFi에서 실행.

### Traditional Convertible Arb
```
전환사채 = 채권 + 주식 콜옵션

매수: 전환사채 (채권 가치 + 내재 옵션 가치)
숏: 동일 기업 주식 (옵션 델타만큼)

수익원:
├── 쿠폰 수익 (채권 이자)
├── 내재 변동성 할인 매수 → 실현 변동성으로 수익
├── 감마 수익 (가격 변동 시 리밸런싱)
└── 크레딧 스프레드 수렴
```

### DeFi Advantage
```
전통 (증권사):
├── 전환사채 차입 어려움
├── 숏 비용 높음
├── 결제 T+2
└── 마진 효율 낮음

DeFi (토큰화):
├── 24/7 거래
├── 자동화된 리밸런싱
├── 즉시 결제
├── 통합 마진 (30% 효율 향상)
└── 플래시론 활용 가능
```

### Return Profile
| 항목 | 수치 |
|------|------|
| 연간 수익 | 8~15% (저변동성) |
| Sharpe Ratio | 1.5~2.5 |
| 최대 손실 | -5~-10% (크레딧 이벤트) |
| 자본 요구 | $100K+ |
| **엣지 수명** | **18개월+** (시장 비효율 지속) |
