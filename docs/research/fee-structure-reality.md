## Core Principle

레버리지 시 **수수료도 포지션 크기에 비례**하지만, **수익도 동일하게 비례**한다.
따라서 마진율(%)은 레버리지와 무관. 절대 금액만 커진다.

단, **고정 비용(가스비)은 비례하지 않으므로** 레버리지가 클수록 고정비 비율은 감소한다.

## What Actually Eats Margin (수수료가 아닌 진짜 비용)

### 1. Funding Rate Direction (최대 변수)

```
강세장: 숏이 수취 (+) → 수익
약세장: 숏이 지불 (-) → 손실

이건 수수료가 아니라 "시장 상태"에 따른 수익/비용.
수수료 대비 크기: 10~100배
```

### 2. KRW ↔ USDC Conversion (진입/퇴장 시 1회)

| 경로 | 스프레드 | $5K 기준 |
|------|---------|---------|
| 은행 송금 | 3~5% | $150~250 |
| 국내 거래소 | 0.3~1% | $15~50 |
| P2P | 0.5~2% | $25~100 |

→ 장기 보유 시 희석됨. 6개월 보유 시 전체 수익의 0.3~1%.

### 3. Slippage (한국 주식 Perp 유동성)

```
BTC/ETH Perp: 슬리피지 ~0.01% (깊은 유동성)
삼성전자 Perp: 슬리피지 ~0.1~0.5% (얕은 유동성)

$25K 포지션: $25~125 슬리피지
1회성 비용.
```

### 4. Liquidation Fee (청산 시에만)

```
Lighter DEX: 포지션의 1% = $250 ($25K 포지션)
→ 청산되지 않으면 비용 $0
→ 보수적 레버리지(3~5x) + 모니터링으로 회피
```

## Honest Monthly P&L ($5K, 5x, Lighter DEX)

### Best Month (풀 강세)
```
수익: $25K × 0.05% × 3 × 30 = +$1,125
비용: 가스 $3
순이익: +$1,122 (자본 대비 +22.4%)
```

### Average Month (강세 70%)
```
강세 21일: $25K × 0.05% × 3 × 21 = +$787.50
약세 9일: $25K × 0.05% × 3 × 9 = -$337.50 (역펀딩)
비용: 가스 $3
순이익: +$447 (자본 대비 +8.9%)
```

### Bad Month (약세 우세)
```
판단: 펀딩 음(-) 전환 감지 → 즉시 청산
손실: 청산 전 역펀딩 2~3일분 = -$75~112
비용: 포지션 청산 가스 $0.10
순손실: -$75~112 (자본 대비 -1.5~2.2%)
```

## Conclusion

**"레버리지 하면 수수료도 비싸지는데 마진이 남나?"**

1. **Lighter DEX에서는 수수료 = $0** → 레버리지 올려도 수수료 증가 없음
2. 타 DEX에서도 수수료는 **진입 1회만** → 보유 중 추가 비용 없음
3. **진짜 마진을 먹는 건 수수료가 아니라 펀딩레이트 방향**
4. 강세장에서 돈 벌고, 약세장에서는 안 하면 됨
5. 현실적 기대: **연 50~100% (평균~좋은 해)**

이전 문서의 "일 0.75% 복리 = 연 500%+"는 풀 강세 365일 가정이므로 비현실적.
정직한 숫자: **강세 70% 기준 연 100% 내외**.
