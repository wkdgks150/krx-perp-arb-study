# Level 3 — Event-Driven & Index Rebalancing Arbitrage

> Last modified: 2026-03-27

---

## Overview

예측 가능한 시장 이벤트(실적 발표, KOSPI 200 리밸런싱, M&A, 정책 변경)가 만드는 가격 비효율을 활용. KRX 폐장 중 이벤트 발생 시 Perp DEX에서 선제적으로 포지션을 잡는 전략.

## Strategy 3-A: Earnings Season Dislocation (실적 시즌 괴리)

### Mechanism
한국 대형주의 분기 실적 발표는 대부분 **장 마감 후 또는 주말**에 이루어진다. Perp DEX는 24/7이므로 즉시 반영되지만, KRX 현물은 다음 개장까지 동결.

### Key Calendar
| 시기 | 이벤트 | 주요 종목 |
|------|--------|----------|
| 1월 말 | Q4 잠정실적 | 삼성전자, SK하이닉스 |
| 4월 말 | Q1 잠정실적 | 삼성전자, SK하이닉스, 현대차 |
| 7월 말 | Q2 잠정실적 | 위와 동일 |
| 10월 말 | Q3 잠정실적 | 위와 동일 |

### Execution
```
삼성전자 Q3 실적 발표: 장 마감 후 금요일 17:00 KST

Case A: 실적 서프라이즈 (+)
- Lighter SAMSUNG Perp: 수 분 내 +3~5% 상승
- KRX: 월요일 09:00까지 동결
- Action: Perp 롱 진입 → 월요일 KRX 개장 전후 청산

Case B: 실적 미스 (-)
- Lighter SAMSUNG Perp: 수 분 내 -3~5% 하락
- Action: Perp 숏 진입 → 월요일 KRX 개장 전후 청산
```

### Return Profile
| 항목 | 수치 |
|------|------|
| 실적 서프라이즈 평균 괴리 | 2~8% |
| 연간 주요 이벤트 | 12~16회 (3종목 × 4분기) |
| 승률 | 65~75% (서프라이즈 방향성 예측) |
| 이벤트당 기대 수익 | 1~4% |
| **연간 기대 수익** | **15~40%** (적극적 참여 시) |

### Risk: 시장 과잉 반응
- Perp에서 이미 과도하게 반영 → KRX 개장 시 되돌림
- 가이던스가 실적보다 중요한 경우 방향 역전

## Strategy 3-B: KOSPI 200 Index Rebalancing

### Mechanism
KOSPI 200은 **분기별 리밸런싱**을 실시한다. 편입/편출 종목 공표 후 ~5영업일의 구현 기간이 존재. 패시브 펀드의 기계적 매수/매도를 선취(front-run).

### Historical Pattern
```
편입 종목: 공표 → 구현일까지 평균 +2~8% 상승 (패시브 매수 압력)
편출 종목: 공표 → 구현일까지 평균 -2~5% 하락 (패시브 매도 압력)
```

### Execution (with Perp DEX)
```
1. KOSPI 200 리밸런싱 공표 (보통 6월, 12월)
2. 편입 예정 종목의 Lighter Perp (있는 경우) 또는 관련 섹터 Perp 롱
3. 편출 예정 종목 숏 (Perp)
4. 구현일 전후 청산
5. Pair로 실행 → 시장 방향성 중립화
```

### Return Profile
| 항목 | 수치 |
|------|------|
| 편입 종목 수익 | +2~8% (5일) |
| 편출 종목 수익 (숏) | +2~5% (5일) |
| Pair 합산 | 4~13% per event |
| 연간 이벤트 | 2~4회 (정기 + 수시) |
| **연간 기대 수익** | **8~25%** |

### Limitation
- Lighter DEX에 해당 종목 Perp이 없으면 직접 실행 불가
- KOSPI 200 지수 Perp으로 간접 헤지 가능

## Strategy 3-C: Macro Event Gap Trading

### Mechanism
글로벌 매크로 이벤트(FOMC, CPI, 지정학적 사건)가 KRX 폐장 시간에 발생할 때 Perp에서 포지션.

### Key Events & Timing
| 이벤트 | 발표 시간 (KST) | KRX 상태 |
|--------|-----------------|---------|
| FOMC 결정 | 03:00~04:00 | **폐장** |
| 미국 CPI | 21:30~22:30 | **폐장** |
| 미국 고용 | 21:30~22:30 | **폐장** |
| 중국 PMI | 10:00~10:30 | 장중 |
| BOK 금리 결정 | 10:00 | 장중 |

### Execution
```
미국 CPI 발표 → 인플레 예상치 상회

22:30 KST: 미국 금리 인상 우려 → 글로벌 리스크오프
Lighter KOSPI 200 Perp: -1~3% 하락
KRX: 익일 09:00까지 동결

Action: KOSPI 200 Perp 숏 진입 (또는 삼성 Perp 숏)
익일 KRX 개장: 갭다운 확인 → 수렴 시 청산
```

### Return Profile
| 항목 | 수치 |
|------|------|
| 매크로 이벤트 괴리 | 0.5~3% |
| 연간 주요 이벤트 | 20~30회 |
| 승률 | 55~65% |
| **연간 기대 수익** | **10~25%** |

### Verdict
**수익성 > 리스크**: YES
- 실적 시즌 괴리(3-A)가 가장 높은 수익/리스크 비율
- 이벤트 캘린더가 공개 정보 → 준비 가능
- **방향성 베팅 요소** 존재 → Level 1보다 리스크 높으나 수익도 높음
- KRX의 긴 폐장 시간이 핵심 엣지
