## Scoring System

6 Dimensions, 각 10점 만점, 총 60점.

| 기준 | 설명 |
|------|------|
| 실현가능성 | 현재 인프라로 즉시 실행 가능한가 |
| 수익성 | 비용 차감 후 기대 순수익률 |
| 리스크 수준 | 원금 손실 가능성 (역수) |
| 자본 효율성 | 필요 자본 대비 수익 |
| 확장성 | 규모 확대 시 수익 유지 |
| 규제 리스크 | 법적 문제 가능성 (역수) |

등급: A (50+), A- (43-49), B+ (40-42), B (37-39), B- (33-36), C+ (26-32), C (20-25)

## Key Insights

### 1. KRX 타임 갭 = 최대 엣지
전 레벨에 걸쳐 **KRX의 80.6% 비운영 시간**이 가장 큰 수익원. 이 갭이 존재하는 한 엣지 지속.

### 2. Lighter DEX = 현재 유일한 실행 플랫폼
삼성/하이닉스/현대차/KOSPI 200 Perp 모두 지원. 제로 수수료. **단일 플랫폼 리스크**이자 **독점적 기회**.

### 3. 2027년 = 전환점
토큰화 증권 시행 시 3중 차익거래 가능. 지금은 Level 1~8로 수익 + 인프라 축적.

### 4. 규제 = 최대 리스크
모든 전략의 공통 취약점. 외환거래법, 가상자산 규제가 게임 룰을 바꿀 수 있음.

### 5. 자동화 = 필수
Level 4 이상은 수동 실행으로는 수익 확보 어려움. 봇 인프라가 수익을 결정.

### 6. Tail Risk Hedge = 보험
총점 최고(43/60)이면서 유일한 "비용" 전략. 하지만 전체 포트폴리오 생존을 보장.
