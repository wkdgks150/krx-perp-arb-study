## 2. Settlement Mechanics

### T+2 결제
- KRX 주식 매매 후 **결제까지 2영업일** 소요
- 실시간 차익거래 시 자본 회전 속도 제약
- Perp DEX는 실시간 결제 → **결제 속도 비대칭**

### Foreign Investor Settlement
- 비거주자: T+6 (SD+4)까지 결제 가능
- 미결제 시 계좌 정지 또는 강제 청산 리스크

### Impact on Arbitrage
```
Day 0: KRX에서 삼성전자 매수 → 결제 Day 2
Day 0: Lighter에서 삼성 Perp 숏 → 즉시 결제

결과: 현물 포지션은 Day 2까지 확정 안 됨
     Perp 포지션은 즉시 오픈
     → 결제 리스크 존재 (T+2 동안 현물 측 불확실)
```

## 3. Short-Selling Restrictions

### Current Status (2026.03)
- **공매도 전면 금지**: 2023년 11월부터 시행, 2025년 3월 30일 이후 재개 논의 중
- **무차입 공매도(naked short)**: 영구 금지
- 기관투자자: 자동 잔고 관리 시스템 의무화

### Impact on Arbitrage
- **베이시스 트레이딩 (Basis < 0 시)**: 현물 숏 불가 → 전략 절반 실행 불가
- **펀딩레이트 차익 (음의 펀딩 시)**: 현물 숏 + Perp 롱 불가
- **대안**: ETF 공매도, 선물 시장 활용 (제한적)

## 4. Capital Controls (자본통제)

### Foreign Exchange Transactions Act (외환거래법)
- 한국에서 해외로의 자금 이동에 엄격한 규제
- 거주자의 해외 투자: 사전 신고 또는 승인 필요
- 일정 금액 이상 해외 송금: 한국은행 보고 의무

### Kimchi Premium Case Study (김치프리미엄)

역사적으로 한국 크립토 거래소와 글로벌 거래소 간 가격 차이(김치프리미엄)가 최대 50%까지 발생했으나, 차익거래가 거의 불가능했던 이유:

1. **자본통제**: 원화 해외 송금 제한
2. **외환거래법**: 무허가 해외 거래소 이용 시 법적 제재
3. **비거주자 제한**: 한국인의 해외 거래소 직접 거래 규제
4. **집행**: 우회 시도자 법적 처벌 사례 다수

### Stablecoin Workaround (스테이블코인 우회)
- USDT/USDC를 통한 글로벌 자본 이동 가능
- 24분 내 가격 수렴 효과 (기존 수일 → 분 단위)
- **한국은행이 식별한 5대 리스크:**
  1. 금융시스템 안정성 영향
  2. 통화정책 전달 효과 저하
  3. 결제시스템 사이버 리스크
  4. **규제 차익 및 자본 유출** ← 가장 관련 높음
  5. 소비자 보호 공백

### Impact on Arbitrage
```
KRX 현물 매수 (원화) → Lighter Perp 숏 (USDC 담보)

이 과정에서:
1. 원화 → USDC 환전 필요 (거래소 경유)
2. USDC를 Lighter DEX로 입금
3. 외환거래법상 보고 의무 가능
4. 대규모 시 한국은행 모니터링 대상
```

## 5. 2027 Tokenized Securities Framework

### Overview
- 한국 정부, 토큰화 증권 합법화 법안 추진 중
- **시행 예정**: 2027년
- **근거법**: 전자증권법 개정
- **시장 규모 전망**: 367조 원 ($249B, BCG 추정)

### Early Movers
- 미래에셋증권: 토큰화 증권 플랫폼 구축 중
- 하나금융그룹: 블록체인 증권 인프라 개발 중

### Impact on Arbitrage
토큰화 증권 시행 시 새로운 차익거래 레이어 등장:

```
Layer 1: KRX 현물 ↔ Perp DEX (현재)
Layer 2: 토큰화 증권 ↔ KRX 현물 (2027~)
Layer 3: 토큰화 증권 ↔ Perp DEX (2027~)

3중 차익거래 가능성
```

### Caveats
- 인가 중개기관을 통한 거래 의무화 가능
- KRX/금융감독원 감독 하에 운영
- 외국인 투자자 접근 제한 유지 가능성

## 6. Tax Implications (참고)

### 현행 세제
- 주식 양도차익: 대주주만 과세 (일반 투자자 비과세, 단 2025년 이후 변경 가능)
- 해외 파생상품 수익: 기타소득으로 과세 (22%)
- 가상자산 소득세: 시행 연기 반복 중 (2027년 시행 예정)

### Arbitrage Tax Structure
```
KRX 현물 매매 수익: 비과세 (대주주 아닌 경우)
Perp DEX 수익: 과세 대상? (법적 지위 불명확)

→ 세법 해석에 따라 총 수익률 크게 변동
→ 전문 세무 자문 필요
```
