# Level 8 — Flash Loan & Cross-Chain Arbitrage

> Last modified: 2026-03-27

---

## Overview

DeFi의 원자적(atomic) 트랜잭션과 크로스체인 브릿지를 활용한 고급 차익거래. **자본 없이(zero-capital)** 또는 **최소 자본으로** 가격 비효율을 포착. 기술적 난이도가 높으나 자본 효율성이 극대화.

---

## Strategy 8-A: Flash Loan Perpetual Arbitrage

### Mechanism
플래시론은 **단일 트랜잭션 내에서** 대규모 자금을 무담보로 차입하고, 차익거래를 실행하고, 차입금을 상환하는 DeFi 메커니즘. 모든 것이 하나의 블록 안에서 원자적으로 실행되므로, 중간에 실패하면 전체가 되돌려진다.

```
단일 트랜잭션 (1 블록):
1. Aave에서 $1,000,000 USDC 플래시론 차입 (수수료 0.05%)
2. DEX-A에서 SAMSUNG Perp 매수 (저가)
3. DEX-B에서 SAMSUNG Perp 매도 (고가)
4. 차익 확보: $3,000 (0.3% 스프레드)
5. Aave에 $1,000,500 상환 (원금 + 수수료)
6. 순이익: $2,500

실패 시: 트랜잭션 전체 revert → 가스비만 손실
```

### Realistic Application to Korean Stocks
```
현재 제한:
- 한국 주식 Perp은 Lighter DEX에서만 거래
- 크로스 DEX 차익거래 불가 (단일 플랫폼)

가능한 변형:
1. Lighter 내 다른 자산 간 상관관계 활용
   - 삼성 Perp 매수 + KOSPI 200 Perp 매도 (지수-종목 괴리)
2. 향후 다른 DEX 한국 주식 추가 시 즉시 활용
3. 스테이블코인 레이어 차익 (USDC/USDT 가격 차이)
```

### Return Profile
| 항목 | 수치 |
|------|------|
| 트랜잭션당 수익 | 0.1~1.0% (차입 금액 대비) |
| 플래시론 수수료 | 0.04~0.09% |
| 가스비 | $0.1~$10 (L2) |
| 순이익 마진 | 0.05~0.9% |
| 필요 자본 | **$0** (가스비만) |
| 일일 기회 | 1~10회 (스프레드 존재 시) |
| **월간 기대 수익** | **$500~$5,000** (봇 운영 시) |

### Technical Requirements
- 스마트컨트랙트 개발 능력 (Solidity)
- MEV 봇 인프라 (Flashbots, private mempool)
- 가스 최적화 기술
- 실시간 가격 모니터링 (다중 DEX)

### Risks
| 리스크 | 심각도 | 설명 |
|--------|--------|------|
| 가스비 > 수익 | 중 | 네트워크 혼잡 시 수익 소멸 |
| 경쟁 봇 | 고 | 전문 MEV 봇과 경쟁 → 스프레드 소멸 속도 빠름 |
| 스마트컨트랙트 버그 | 고 | 봇 코드 결함 → 자금 손실 |
| 엣지 수명 | 짧음 | 2~3개월 후 경쟁에 의해 수익 소멸 |

## Strategy 8-B: Cross-Chain RWA Arbitrage

### Mechanism
동일한 RWA 토큰이 다른 블록체인에서 다른 가격으로 거래되는 경우 크로스체인 브릿지를 통해 차익거래.

```
연구 데이터:
- 9개 체인에서 260,000+ 크로스체인 차익거래 실행 관측
- 연간 수익: $9.5M (전체 시장)
- 브릿지 수수료: 0.1~1%
```

### Execution
```
Scenario: 토큰화 삼성전자 (향후)

Ethereum 가격: $50,000
Arbitrum 가격: $50,300 (0.6% 프리미엄)

1. Ethereum에서 토큰화 삼성 매수 @ $50,000
2. 브릿지로 Arbitrum에 전송 (수수료 0.1%, 시간 ~10분)
3. Arbitrum에서 매도 @ $50,300
4. 순이익: $250 per unit (0.5%)
```

### Current Limitations
- 한국 주식 토큰이 아직 다중 체인에서 거래되지 않음
- 크로스체인 브릿지 해킹 리스크 높음 (역대 $2B+ 손실)
- 향후 2027년 토큰화 증권 프레임워크 이후 기회 확대

### Return Profile
| 항목 | 수치 |
|------|------|
| 크로스체인 스프레드 | 50~500 bps |
| 브릿지 수수료 | 10~100 bps |
| 순 마진 | 10~100 bps |
| 일일 기회 | 수 회 (자산 존재 시) |
| **연간 기대 수익** | **5~15%** (자동화 시) |

## Strategy 8-C: Bridge-Optimized Spot-Perp Arbitrage

### Mechanism
기존 현물-Perp 차익거래를 크로스체인 브릿지 최적화로 비용 절감. 이더리움 메인넷 가스비($10~50)를 L2 브릿지 활용으로 near-zero까지 절감.

```
기존 방식:
Ethereum 현물 매수 + Ethereum Perp 숏
가스비: $20~50 per trade
최소 스프레드: 0.5% (가스비 커버)

최적화:
L2 (Arbitrum/Lighter) 현물 매수 + Lighter Perp 숏
가스비: $0.1~$1 per trade
최소 스프레드: 0.02% (가스비 커버)
→ 기존에 불가능했던 타이트한 스프레드도 수익화 가능
```

### Return Profile
| 항목 | 수치 |
|------|------|
| 사이클당 수익 | 0.1~0.5% |
| 일일 사이클 | 5~20회 (자동화) |
| 월간 수익률 | 4~20% |
| **연간 기대 수익** | **30~60%** (최적 조건) |

## Combined Evaluation (Level 8)

| 기준 | 8-A (Flash Loan) | 8-B (Cross-Chain) | 8-C (Bridge Opt) |
|------|-----------------|-------------------|-----------------|
| 실현가능성 | 5 | 3 | 7 |
| 수익성 | 7 | 6 | 8 |
| 리스크 수준 | 6 | 4 | 6 |
| 자본 효율성 | 10 | 6 | 7 |
| 확장성 | 3 | 3 | 4 |
| 규제 리스크 | 5 | 4 | 6 |
| **총점** | **36/60 (B-)** | **26/60 (C+)** | **38/60 (B)** |

### Verdict
**수익성 > 리스크**: 8-A는 CONDITIONAL, 8-B는 FUTURE, 8-C는 YES
- Flash Loan(8-A): 자본 효율 극대화이나 **기술 진입장벽 최고** + 경쟁 극심
- Cross-Chain(8-B): 2027년 이후 본격화. 현재는 브릿지 리스크 > 수익
- Bridge Optimization(8-C): 가장 실용적. 가스비 절감으로 타이트한 스프레드 수익화
- **Level 8은 개발자/엔지니어에게 유리한 레벨** — 코딩 역량이 직접 수익으로 전환
