## Strategy 1: Funding Rate Arbitrage (펀딩레이트 차익거래)

### Mechanism
Perpetual futures는 만기가 없으므로, **펀딩레이트(funding rate)**라는 주기적 지불 메커니즘으로 perp 가격을 현물 가격에 수렴시킨다.

- **양(+) 펀딩레이트**: Perp > Spot → 롱 포지션이 숏 포지션에 지불
- **음(-) 펀딩레이트**: Perp < Spot → 숏 포지션이 롱 포지션에 지불
- **주기**: 대부분 8시간마다 (하루 3회)

### Execution
```
1. 현물 시장에서 삼성전자 매수 (Long Spot)
2. Lighter DEX에서 삼성전자 Perp 매도 (Short Perp)
3. 델타 = 0 (시장 방향성 무관)
4. 양(+) 펀딩레이트 → 숏 포지션이 펀딩비 수취
5. 8시간마다 수익 누적
```

### Return Profile
| 시장 상황 | 8h 펀딩레이트 | 연간 수익률 |
|-----------|---------------|-------------|
| 평상시 | 0.015% | ~19% |
| 강세장 | 0.05~0.2% | 60~115% |
| 약세장 | -0.01~0% | 마이너스 (주의) |

### Example Calculation
```
자본: $100,000
포지션: 삼성전자 현물 $50,000 + Perp 숏 $50,000
펀딩레이트: +0.015% per 8h
일일 수익: $50,000 × 0.015% × 3 = $22.50
연간 수익: $22.50 × 365 = $8,212.50 (16.4%)
필요 마진: Perp 숏 측 ~$10,000 (10x 기준)
```

### Pros
- 시장 방향 무관 (delta-neutral)
- 꾸준한 수익 흐름 (패시브 인컴)
- Lighter DEX 제로피 → 수수료 부담 없음

### Cons
- 펀딩레이트 반전 시 손실 (약세장)
- 양측 자본 묶임 (자본 효율 50% 수준)
- 가격 급변 시 숏 포지션 청산 위험

## Strategy 2: Basis Trading (베이시스 트레이딩)

### Mechanism
현물과 선물 간 가격 차이(basis)가 발생할 때, 이 스프레드를 수익화한다.

- **Basis = Perp Price - Spot Price**
- Basis > 0 (contango): 현물 매수 + Perp 매도
- Basis < 0 (backwardation): 현물 매도 + Perp 매수

전통 선물은 만기에 수렴하지만, perp은 만기가 없으므로 **펀딩레이트가 수렴 메커니즘** 역할.

### Execution
```
1. Basis 모니터링 (Lighter SAMSUNG perp vs KRX 삼성전자 현물)
2. Basis > 임계값 (예: 0.5%) → 현물 매수 + Perp 매도
3. Basis 수렴 시 양쪽 동시 청산
4. 스프레드 차이만큼 수익
```

### Return Profile
| 항목 | 수치 |
|------|------|
| 일반 베이시스 범위 | 0.5~2% |
| 회전 주기 | 수 시간~수 일 |
| 한국 주식 기대 스프레드 | 0.5~1.5% (아시아 시장 시간대) |

### Key Difference from Strategy 1
펀딩레이트 차익거래는 **시간에 비례한 수익**, 베이시스 트레이딩은 **스프레드 크기에 비례한 수익**.

### Pros
- 명확한 수익 목표 (스프레드)
- 펀딩레이트 방향 무관

### Cons
- 스프레드가 확대될 수 있음 (수렴까지 시간 불확실)
- 양측 동시 실행 어려움 (슬리피지)
- 유동성 부족 시 대규모 실행 불가

## Strategy 3: Off-Hours Price Dislocation (장외시간 가격 괴리)

### Mechanism
KRX의 가장 독특한 특성을 활용하는 전략. KRX가 닫혀 있는 동안(하루 17.5시간, 주말 65시간) 가격에 영향을 미치는 뉴스/이벤트가 발생하면:

- **Perp 시장**: 즉시 가격 반영 (24/7)
- **현물 시장**: 다음 개장까지 동결

### Execution Scenarios

#### Scenario A: 장후 악재
```
금요일 20:00 KST: 삼성전자 실적 미스 발표
Lighter SAMSUNG perp: -5% 하락
트레이더: Perp 숏 진입 @ 새 가격
월요일 09:00 KST: KRX 개장 → 삼성 현물 -5% 갭다운
트레이더: 현물 매수 + Perp 숏 커버 → 수익 확보
```

#### Scenario B: 장후 호재
```
화요일 18:00 KST: SK하이닉스 HBM 수주 발표
Lighter SKHYNIX perp: +3% 상승
트레이더: Perp 롱 진입
수요일 09:00 KST: KRX 개장 → SK하이닉스 현물 +3% 갭업
트레이더: Perp 롱 청산 → 수익 확보
```

### Key Windows
| 구간 | 갭 시간 | 빈도 |
|------|---------|------|
| 매일 장후~익일 장전 | 17.5시간 | 매일 |
| 금요일 장후~월요일 장전 | 65시간 | 매주 |
| 공휴일 | 24~96시간 | 연 15~20일 |

### Return Profile
- 이벤트 기반 → 일정한 수익률 산출 불가
- 이벤트당 1~5% 수익 가능
- 연간 주요 이벤트 횟수: 실적시즌(4회) + 매크로 이벤트(수시)

### Pros
- 가장 직관적인 "시간 차이" 활용
- 자본 효율 높음 (이벤트 때만 자본 투입)
- KRX 특유의 구조적 기회

### Cons
- 방향성 베팅 (delta-neutral 아님)
- 뉴스 반응이 예상과 다를 수 있음
- Perp 유동성 부족 시 슬리피지 큼
- 장 열릴 때 현물 실행 지연 가능

### Why This is NOT "Risk-Free"
방향성 포지션이므로 가격이 예상과 반대로 움직이면 손실. 뉴스 해석 오류, 시장 과잉 반응, 오라클 지연 등이 리스크.

## Strategy 4: Cross-Market Arbitrage (크로스마켓 차익거래)

### Mechanism
동일 자산이 복수의 perp-DEX에서 거래될 때, 플랫폼 간 가격/펀딩레이트 차이를 활용.

### Execution
```
가상 시나리오 (향후):
Platform A (Lighter) 삼성 perp 펀딩: +0.02% per 8h
Platform B (미래 플랫폼) 삼성 perp 펀딩: +0.01% per 8h

→ Platform B에서 롱, Platform A에서 숏
→ 0.01% × 3/일 = 3.65% 연간 스프레드 수익
```

### Current Status
- 한국 주식 perp 지원 플랫폼: **Lighter DEX만** (2026.03 기준)
- 크로스마켓 차익거래: **현재 불가**
- 향후 다른 DEX가 한국 주식 추가 시 가능

### Future Potential
- 추가 연 3~5% 수익 가능
- 낮은 상관관계 → 포트폴리오 분산 효과
- 복수 DEX 인프라 구축 필요

### Pros
- Delta-neutral
- 한국 주식 DEX 생태계 확장 시 자동 기회 확대

### Cons
- **현재 실행 불가**
- 복수 플랫폼 마진 관리 복잡
- 가스비/브릿지 비용
- 오라클 불일치 리스크

## Strategy 5: Delta-Neutral Volatility Strategy (델타뉴트럴 변동성 전략)

### Mechanism
한국 주식은 일반적으로 BTC/ETH보다 변동성이 낮다. 그러나 크립토 트레이더들이 높은 레버리지를 사용하면 펀딩레이트가 상승한다. 이 "레버리지 프리미엄"을 방향성 리스크 없이 수취.

### Execution
```
1. Lighter DEX에서 한국 주식 Perp의 펀딩레이트 모니터링
2. 펀딩레이트 > 임계값 (예: 0.02%) 유지 시:
   - 현물 매수 + Perp 숏 (또는 inverse 포지션)
3. 펀딩레이트 < 임계값 시 포지션 청산
4. "변동성을 파는" 전략 — 레버리지 수요가 높을 때 수익
```

### Strategy 1 vs Strategy 5 차이
| 항목 | Strategy 1 (펀딩레이트 차익) | Strategy 5 (델타뉴트럴 변동성) |
|------|------------------------------|-------------------------------|
| 진입 기준 | 항상 유지 | 펀딩레이트 임계값 초과 시만 |
| 보유 기간 | 장기 | 단기~중기 (조건부) |
| 수익 프로필 | 꾸준 | 높은 펀딩 시 집중 수익 |
| 자본 효율 | 낮음 (항상 묶임) | 높음 (선택적 투입) |

### Return Profile
| 시장 상황 | 예상 수익 |
|-----------|-----------|
| 강세장 (높은 레버리지) | 20~40% 연간 |
| 보합장 | 10~15% 연간 |
| 약세장 (낮은 레버리지) | 포지션 미진입 → 0% |

### Pros
- 조건부 진입 → 자본 효율 극대화
- 한국 주식의 낮은 변동성 → 청산 리스크 감소
- 시장 방향 무관

### Cons
- 임계값 판단 주관적
- 펀딩레이트 변화 속도 > 포지션 조정 속도 가능
- 비 투자 기간 동안 기회비용
